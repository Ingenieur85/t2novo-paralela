## Como Usar

Para compilar, faça:
```
make
```

Para rodar todos as partes A e B (usado para a planilha), faça:
```
./run.sh
```
o script run.sh vai chamar o roda-todos-slurm.sh para cada um dos tamanhos e vai salvar os resultados em um arquivo .out
